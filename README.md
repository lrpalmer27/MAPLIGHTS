# LIGHT UP MAP PROJECT
This project is based on this project, which I found on Facebook. I believe they are commercially available somewhere. I wanted to add Canada, and a select few cities that are meaningful to me.
![INSPO](Inspo.png.png)

## Basis
This project is based on the Metostat library of maintained weather data across the globe.

Check out Metostat:
    https://meteostat.net/en/

Starting set of north american city coordinates curtosy of: https://www.infoplease.com/us/geography/latitude-and-longitude-us-and-canadian-cities

Futher edits and functionality beyond these two sources are built by yours truly.
